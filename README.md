## 搬瓦工"单机房套餐"是什么意思？

搬瓦工大部分套餐购买后，可以在 KiwiVM 控制面板里自由迁移机房——从洛杉矶换到日本，再换到荷兰，都不收费，只消耗流量。这个功能是搬瓦工的一大卖点，很多人冲着这个买的。

**搬瓦工单机房套餐**，就是那些购买后被锁定在某一个特定机房、无法通过后台迁移到其他机房的套餐。机房固定，IP 不随便变，也无法享受"买错机房可以换"的灵活性。

为什么会有这类套餐？原因很直接：搬瓦工用单机房限制换来了更低的价格。常规 CN2 GIA-E 套餐最便宜年付 $169.99，而同线路、类似配置的单机房限量版只要 $38-$53 左右。价格差到这个程度，代价就是你得接受机房绑定。

## 哪些情况会导致搬瓦工"无法迁移机房"？

不是所有迁移失败都是"单机房套餐"造成的。大概有三种情况：

1. **套餐本身不支持迁移**。这是真正的"搬瓦工单机房套餐"，产品页面会明确写 Location Only 或者购买时机房选项只有一个。
2. **目标机房已满**。提示 "Region is full"，不是你的套餐有问题，等一段时间或换个目标机房即可。
3. **VPS 的 IP 被墙**。IP 出问题时后台会禁用迁移功能，需要先处理 IP 问题（换 IP 或等待）。

如果你打开 KiwiVM 后台，发现 "Migrate to another DC" 选项压根没有，或者可选机房列表是空的，那八成是第一种情况——你买的就是单机房套餐。

## 单机房限量版 vs 可迁移常规套餐：该怎么选？

这是真正的核心问题。

先说价格差距，以洛杉矶 CN2 GIA 线路为例：常规 CN2 GIA-E 套餐最低 $169.99/年，The DC9 Plan 单机房版只要约 $38/年，同类线路差了四五倍。MegaBox Pro 是 $49/年拿 2 核 2GB 40GB 2TB 流量，换成常规套餐同配置得 $299.99/年——能省下 250 刀。

性价比差距这么大，为什么不是所有人都买限量版？

**选限量版单机房套餐，适合这些情况**：
- 你已经用过某个机房，知道那个机房对你的网络环境效果好
- 你的使用场景对机房稳定性有要求，不打算频繁换 IP
- 你能接受限量、抢购、偶尔缺货的节奏
- 目标就是低价拿到一台固定用途的 VPS（建站、跑脚本、挂服务等）

**选常规多机房套餐，适合这些情况**：
- 第一次买搬瓦工，不确定哪个机房对自己最合适
- IP 被墙的情况时有发生，需要随时切换机房换 IP
- 想同时访问多个地区的资源，机房灵活性更重要
- 不想等补货，需要即时购买

不夸张地说，如果你已经摸清楚了自己的用网情况，单机房限量版基本上是性价比的天花板。问题是很多人在没搞清楚之前就买了，要么觉得线路不对，要么后来 IP 出问题想换机房发现换不了。

## FAQ

**Q：搬瓦工单机房套餐 IP 被墙了怎么办？**

A：正常套餐 IP 被墙后，可以通过迁移机房换 IP。但单机房套餐不支持迁移，只能用付费换 IP（大概 $8.79/次）或者等 IP 解封。这是单机房套餐最大的风险点，买之前需要考虑进去。

**Q：搬瓦工单机房限量版没货怎么办？**

A：等补货，或者直接买常规套餐。限量版补货时间不固定，有时候几周，有时候几个月，搬瓦工官方网站会更新库存状态。有即时需求建议直接选常规套餐，不必卡着限量版。

**Q：KVM 套餐和 CN2 GIA-E 套餐都可以迁移，区别在哪？**

A：KVM 套餐能迁移到的机房都是普通线路（DC2/DC4/DC8 等），国内访问速度一般；CN2 GIA-E 套餐可以迁移到 DC6/DC9（CN2 GIA 线路）、日本软银、荷兰联通 EUNL_9 等高端机房，国内速度快很多。预算允许的情况下，CN2 GIA-E 值得多花那份钱。

**Q：搬瓦工的 30 天退款政策对单机房套餐也适用吗？**

A：适用。搬瓦工支持新账户 30 天内无理由全额退款，包括限量版单机房套餐，满足退款条件后在后台直接提交申请，一般自动处理。

**Q：单机房套餐续费价格会变吗？**

A：不会。搬瓦工的续费价格和首购价格一致，不存在"低价吸引首购、高价续费"的情况，这点是用过的人普遍认可的。
